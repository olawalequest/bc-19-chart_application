# Chart Application

# INTRODUCTION
Chart Application is a Chart apllication with the following features: 
 - Plots a chart based on data supplied.
 - The application has the four basic chart forms:
    . Pie Chart
   . Histogram
   . Bar Chart
   . Line Chart
  
# Tools Used for development
 - HTML5
 - css
 - Javascript
 - Jquery 3.1.1
 
# Supported Chart Forms
 - Pie Chart
 - Histogram
 - Bar Chart
 - Line Chart
